How Do I Connect via Emulation?

FOR WINDOWS ONLY


.exe Method
You must first have Snes9x installed
installed, and then you need to have the BS-X BIOS Rom
in the root of your Snes9x folder.
You must then download Satellaview+ exe file and
place it into the root of the Snes9x emulator.
Then download the Git Portable Installer
and run the .exe file. When you
run it, when it asks for a directory
to add the Git installer to, you
should make a folder in the Snes9x
directory called "git", and then
copy the URL going to that folder
and paste it into the Git installer.
Then, run the .exe file, and
if everything worked, you should
see a "satdata" folder in your
Snes9x directory. Now you can
run the emulator and experience
Satellaview+!
