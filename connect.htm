How Do I Connect via Emulation? (SNES9X)

FOR WINDOWS ONLY


Download the Emulator packaged with the BS-X rom here,
upzip the folder and then after you
place the folder wherever you please,
download the Satellaview+ Client with the Auto-Updater and
unzip it and place it into
the root of the emulator folder.
After that, run the .exe file, and if everything worked
properly, you should get a successful
confirmation and there should be a
"satdata" folder near the .exe file.
You can now load up the
emulator and enjoy Satellaview+!


How Do I Connect via Emulation? (BSNES-PLUS)


FOR WINDOWS ONLY


Download the Emulator packaged with the BS-X rom and the Satellaview+ Client and the Auto-Updater.
Then upzip the folder and then
place the folder wherever you please.
After that, run the Satellaview+ Client.exe file, and if everything worked
properly, you should get a successful
confirmation and there should be a
"bsxdat" folder.
You can now load up the
emulator and go to "System", and then
go to "Special Cartridge", and then go
to "Load BS-X Cartridge".
Then for the base acrtridge go
(in the emulator folder) to "BSX" and choose
the BS-X.sfc file. after that
click load and enjoy Satellaview+!


How Do I Update?

There are a couple ways this can be done,
the most efficient way to do so is to use the Auto-Update.bat file bundled in with the
client installer for SNES9X and in the main and only download link for BSNES-Plus installs (But here it is if you need it for some reason).
This .bat file will just have the Satellaview+ client infinitely run
in the background so you don't have to worry
about loading up the .exe file any time you
need to update.
Alternatively you can just run the .exe file
at the times of 8am (8), 11am (11), 2pm (14), 5pm (17), 8pm (20), and 11pm (23)
and that would also
work, but it'd be tedious.



How Do I Connect on a SuperDisc+?

All you have to do is download the BS-X iso file here
and burn it to a CD, then just plug the
SuperDisc+ into ethernet or alternatlively
use a Android phone to share the WiFi (if the phone
supports it) via USB Tethering, and then just run the disc and
it should then load the Satellaview+ client and then load the
BS-X rom and then you should connect to Satellaview+!
Real-time updates should also work out of the box, so
no need to worry about setting up Task Scheduler or running
the Satellaview+ client again

How Do I Connect on a Super Famicom?
