How Do I Connect via Emulation?

FOR WINDOWS ONLY


.exe Method
You must first have Snes9x installed
installed, and then you need to have the BS-X BIOS Rom
in the root of your Snes9x folder.
You must then download Satellaview+ exe file and
place it into the root of the Snes9x emulator.
The run the .exe file, and if
everything worked, you should have
a "satdata" folder in your snes9x root.
Now you can start playing!
