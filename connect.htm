How Do I Connect via Emulation?

FOR WINDOWS ONLY


.exe Method
Download the Emulator packaged with the BS-X rom here,
upzip the folder and then after you
place the folder wherever you please,
download the Satellaview+ Client and
unzip it and place it into
the root of the emulator folder.
After that, go into the client folder
and scroll until you see "Satellaview+.exe".
Run the .exe file, and if everything worked
properly, you should get a successful
confirmation and there should be a
"satdata" folder near the .exe file.
Take the "satdata" folder and drag it
out to the root of the emulator and
place it there. You can now load up the
emulator and enjoy Satellaview+!
